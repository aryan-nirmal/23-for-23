## 🛠️ Architecture

Instead of managing 23 separate Git repositories, this is structured as a **Monorepo**. Each project is fully self-contained with its own `package.json`, dependencies, and build pipeline.

To run a specific project locally, simply navigate to its folder and follow the project-specific README.
